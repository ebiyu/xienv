Usage: xienv <command> [<args>]

Some useful xienv commands are:
   init        Configure the shell environment for xienv
   global      Set or show the global Vivado/Vitis version(s)
   local       Set or show the local application-specific Vivado/Vitis version(s)
   version     Show the current Vivado/Vitis version(s) and its origin
   versions    List all Vivado/Vitis versions available to xienv

For full documentation, see: https://github.com/ebiyu/xienv#readme
